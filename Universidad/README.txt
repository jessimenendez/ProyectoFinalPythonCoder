Abrir la carpeta Universidad y ejecutar el comando python manage.py runserver
Se trata de una aplicacion que maneja Cursos, Alumnos y Profesores.
A través de los diferentes formularios se puede agregar datos a los tres modelos.
Se puede Editar o Eliminar un Curso.
Se puede buscar un curso en la seccion de Cursos.
Se puede navegar a traves de la barra de navegación.
